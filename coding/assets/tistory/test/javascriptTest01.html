{
    function solution(arr, divisor) {
        var answer = [];
        
        const div = arr.filter( el => el % divisor == 0 );    
        answer = div.length > 0 ? div.sort( (a, b ) => a - b ) : [-1];
        
        return answer
    }
            
    document.write(solution([1,2,3,4,5], 2))
}

1. filter로 2로 나눴을 때 나머지값이 0인 (즉 2의 배수인) 값을 골라냄 == 2, 4
2. div.length(2) > 0이 참임 : 참이면 div안의 순서를 a-b 순서로 정렬하고 거짓이면 -1 출력
3. a-b 순서대로 [2, 4] 출력됨
