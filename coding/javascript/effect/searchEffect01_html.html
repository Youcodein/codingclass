HTML 태그 검색하기

  • a : a 태그는 다른 페이지 이동을 설정합니다.
  • abbr : abbr 태그는 줄임말이나 머리글자를 표현하는데 사용합니다.
  • acronym : acronym 태그는 줄임말이나 머리글자를 표현하는데 사용합니다.
  • address : address는 가까운 HTML 요소의 사람, 단체, 조직 등에 대한 연락처 정보를 나타냅니다.
  • applet : applet는 자바 애플릿을 보이게 하는 곳에 쓰입니다.
  • area : area는 이미지의 핫스팟 영역을 정의하고 하이퍼링크를 추가하는데 쓰입니다.
  • article : article는 문서, 페이지, 애플리케이션, 또는 사이트 안에서 독립적으로 구분해 배포하거나 재사용할 수 있는 구획을 나타낼때 사용합니다.
  • aside : aside는 무서의 주요 내용과 간접적으로 연관된 부분을 나타낼때 사용합니다.
  • audio : audio는 문서에 소리 콘텐츠를 포함할 때 사용합니다.
  • b : b는 글씨를 굵게 할 때 사용합니다. 그러나 를 사용해 텍스트를 꾸미면 안됩니다. 대신 CSS font-weigh를 사용하여 굵은 글씨를 나타내거나 strong으로로 중요한 텍스트를 표현하세요.
