01. 숫자 메서드 : Number.MAX_VALUE

Number.MAX_VALUE 속성은 JavaScript가 표현할 수 있는 제일 큰 양의 숫자 값을 나타냅니다

02. 숫자 메서드 : Number.MIN_VALUE

Number.MIN_VALUE 속성은 JavaScript가 표현할 수 있는 제일 작은 양의 숫자 값을 나타냅니다

03. 숫자 메서드 : Number.MAX_SAFE_INTEGER

Number.MAX_SAFE_INTEGER 상수는 JavaScript에서 안전한 최대 정수값을 나타냅니다.

04. 숫자 메서드 : Number.MIN_SAFE_INTEGER

Number.MIN_SAFE_INTEGER 상수는 JavaScript에서 안전한 최소 정수값을 나타냅니다.

05. 숫자 메서드 : Number.NaN

Number.NaN 속성은 Not-A-Number(숫자가 아님)를 나타냅니다. NaN과 같습니다.
